Pour réussir cet exercice, j'ai fais un "man ls" et j'ai lu chaque attribut:

-l est un attribut permettant d'avoir un format listé 
-h est un attribut qui sert à afficher les fichiers lisibles 
-a est un attribut permettant d'afficher les fichiers cachés
